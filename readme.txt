AutoShoop plugin is a simple module to allow WordPress website to communicate with Autoshop Server and user's mobile device.

== Description ==

AutoShop implements the most important part of eCommerce system:

* Product & Sale Website (integrate AutoShop SDK or AutoShop WordPress Plugin)
* Sale & Order & Report Management Server.
* AutoShop app: the client mobile application for employee and manager.

AutoShop WordPress Plugin helps the shop owner (manager) who using WooCommerce Plugin to manage their orders and employee using notification system whenever customer order any product on website.

<a href="https://autoshop.aucoz.com" target="_blank">AutoShop by Aucoz Support</a>

= Features =

* Send order's notification to server in order to broadcast to all linked-devices.

= Information we collect =

AutoShop plugin requires necessary infomations from WooCommerce in order to work with our AutoShop Server.

The types of information collected are:

* Order data: date, status, close-date, link.
* Customer data: name, phone, address.
* Product data: name, price, quantity.

= The way we use information =

We use the information you provide about your order when managing your order on AutoShop App. We do not share this information with outside parties except to the extent necessary to complete that process.

== Installation ==

1. Deactivate the plugin if you have the previous version installed.
2. Extract "autoshop.zip" archive content to the "/wp-content/plugins/autoshop" directory.
3. Activate "AutoShop" plugin via 'Plugins' menu in WordPress admin menu.
4. Enter secret key in AutoShop settings page.
5. That's it. You're ready to go!

== Frequently asked questions ==

= What is secret key? =

Secret key is the token used to communicate with server from website.

= Where can I find secret key? =

Install AutoShop App (iOS/Android) > create account > create shop > view shop detail and then generate secret key.

For more detail please refer <a href="https://autoshop.aucoz.com">autoshop.aucoz.com</a>
